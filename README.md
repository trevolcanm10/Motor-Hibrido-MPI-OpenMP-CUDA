# Motor Híbrido MPI + OpenMP + CUDA

Motor híbrido de alto rendimiento orientado al análisis eficiente de logs NASA HTTP,
combinando paralelismo multinodo (MPI), multinúcleo (OpenMP) y aceleración GPU (CUDA).

## Tecnologías usadas

- MPI (Message Passing Interface)
- OpenMP (Paralelismo en CPU)
- CUDA (Aceleración en GPU)
